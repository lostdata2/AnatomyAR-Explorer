## О проекте

**Анатомия человека в дополненной реальности** - это инновационное образовательное приложение, разработанная для android, которое превращает изучение анатомии человека в увлекательное интерактивное путешествие с использованием технологий дополненной реальности.

**Ключевые преимущества:**
- **Интерактивные 3D-модели** органов человеческого тела в реальном масштабе
- **AR-режим** для наложения анатомических структур на реальное пространство
- **Детализированные системы организма**: скелетная, мышечная, кровеносная, нервная и другие
- **Образовательный контент** с проверенной медицинской информацией

Проект создан для преподавания уроков анатомии в младших классов и для всех, кто интересуется устройством человеческого тела и технологиями дополненной реальности.

## 🛠 Используемые технологии

- **Unity** 6000.2.7f2
- **AR Foundation**
- **Google ARCore**
- **Blender** (Для создания 3-d моделей)
- **Android SDK/NDK** (для сборки проекта под целевую архитектуру)

## ⚙️ Минимальные системные требования
- Android **API level 21** (android 5.0)
- Поддержка **Google ARCore**
- **Задняя камера** с поддержкой HDR
- Процессор на архитектуре **ARM64**
- **2GB** RAM

## 🚀 Использование
## Основной функционал
- **Режим исследования**: Изучайте отдельные системы организма
    
- **AR-режим**: Проецируйте 3D-модели в реальное пространство
    
- **Анатомический атлас**: Подробная информация о каждом органе

### Установка
1. **Скачайте** последний [Realese](https://github.com/lostdata2/AnatomyAR-Explorer/releases)
2. **Установите** на ваш android-смартфон

### Работа с приложением
1. **Откройте** приложение
2. **Выберите** класс для начала работы (можно изменить позже в настройках)
3. На главном экране **нажмите** кнопку старт
4. **Виберите** тип поверхности (стены и пол/стол)
5. Затем **отсканируйте** повехность с нескольких ракурсов, пока выбранная поверхность не отсканируется
7. **Выберите** орган в меню органов
8. **Нажмите** в пределах отсканированной области чтобы поставить орган на поверхность
9. Вы можете его **рассматривать** с разных сторон перемещая камеру относительно модели органа (в будущем будет поддержка жестов)

## ⌨︎ Разработка

1. Установите **Unity** 6000.2.7f2

2. Клонируйте репозиторий
```bash
git clone https://github.com/lostdata2/AnatomyAR-Explorer
```

3. Откройте проект в Unity (кнопка "add from disk" в unityhub)
    
4. Внесите необходимые правки в проект.

5. Соберите проект (Files >> Build settings >> build)
